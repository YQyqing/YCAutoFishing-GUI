# 更新日志

## [1.0.6-GUI] - 2024-12-15

### 新增功能
- ✨ 添加图形化配置界面（按 `O` 键打开）
- ✨ 添加右下角弹出式状态提示（绿色表示开启，红色表示关闭）
- ✨ 添加配置文件支持（JSON 格式，位于 `config/yc_auto_fishing.json`）
- ✨ 添加配置热加载功能（可在游戏内重新加载配置文件）
- ✨ 添加多语言支持（中文/英文）
- ✨ 添加 HUD 显示开关配置

### 修复问题
- 🐛 修复进入服务器时的崩溃问题（NullPointerException）
- 🐛 修复自动钓鱼无法正常工作的问题
- 🐛 修复配置加载失败时的错误处理

### 改进
- 🔧 优化代码结构，提高稳定性
- 🔧 改进错误处理，增加空值检查
- 🔧 优化 HUD 显示逻辑，减缓动画速度
- 📝 完善文档和注释

### 配置项
- `autoFishingEnabledByDefault`: 进入游戏时是否默认开启自动钓鱼
- `stopDistance`: 文本牌跟随停止距离（默认 2.0）
- `hudEnabled`: 是否显示 HUD 提示（默认 true）
- `stuckTickThreshold`: 卡住检测阈值，单位 tick（默认 200）
